# Cursor Setup Portfolio Project
## Objective:
This project documents the setup process for Cursor IDE, GitHub repository creation, and troubleshooting encountered during the installation and configuration process.
## Tools Installed
- Cursor IDE Version (3.9.16).
- GitHub.
- Cursor Marketplace.
## Steps Completed
- Installed Cursor IDE.
- Created and logged into Cursor account.
- Explored the Cursor marketplace and plugin system.
- Created a GitHub account.
- Created a public GitHub repository.
- Added a README.md file.
- Documented the setup process and findings.
## Issues Encountered
### Issue 1: Extensions Menu Not Available
The assignment instructions mentioned an "Extensions" menu inside Cursor.
Upon following the instructions I found that there was no option availabe by the name "Extensions" and rather the new version uses a new marketplace and Plugin system.  
#### Resolution
I explored the updated interface and located the Marketplace through the "Customize" section.
### Issue 2: Claude Code and Codex Plugins Not Found
The assignment instructed me to search and download Claude Code and Codex Plugins. 
After searching the current Cursor Marketplace for those plugins I found that neither one was available. 
#### Resolution
I investigated the updated Marketplace, verified the search results, documented the discrepancy, and continued with the remaining setup process using the latest Cursor environment.
## Outcome
Successfully completed the setup process, created a public GitHub repository, and documented the installation workflow along with troubleshooting findings.
